Pasos para utilizar el LOG IN

  1. Ingresar nombre de usuario el campo de texto así como se observa en la siguiente imagen.

       



Imagen 2 - CONTROL DE ACCESO

Obs.: El funcionario debe tener en cuenta no sobrepasar la longitud requerida para el nombre de usuario.

  1. Ingresar la contraseña en el campo de texto así como se observa en la siguiente imagen.






Imagen3 - CONTROL DE ACCESO

  1. Una vez ingresado el nombre de usuario y la contraseña se procede a presionar sobre el botón

Imagen 4 - Botón Ingresar


Obs1.: Si los datos ingresados son correctos el sistema automáticamente va a cerrar el formulario LOG IN y mostrara el MENU PRINCIPAL como se observa en la imagen 6, caso contrario que los datos ingresados no existan en la base de datos automáticamente el sistema nos mostrara una ventana de validación como se observa en la imagen 4.









         Imagen 5 – MENSAJE DE ADVERTENCIA

Obs2.: En el caso de que algún campo ya sea nombre de usuario o clave se deje vacío y se haga clic al botón de Ingresar, le aparecerá una ventana con un mensaje de advertencia cómo observamos en la imagen  6










Imagen 6 - MENSAJE DE ADVERTENCIA

  1. Si todos los datos ingresados son correctos, se tendrá acceso al sistema, ingresará al Menú Principal.

Obs.: Aquí le damos a conocer los formularios del sistema, que sólo podrán ser ingresados a través de los usuarios activos.
